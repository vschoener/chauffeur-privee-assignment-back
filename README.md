# Loyalty API

Backend assignment that use:
- Node
- Flow
- RabbitMQ
- MongoDB

## Requirements
- Having a RabbitMQ running

## Installation
``` bash
> npm install
> npm run build
```

## Settings

Setting up the .env if required (Mandatory with Docker usage)
```bash
> cp .env.sample .env
```

## Dev
```bash
> npm watch
```

## Launch tests
``` bash
> npm test
```

## Start server
``` bash
> npm start
```
